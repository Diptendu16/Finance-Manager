Expense Tracker

A fully responsive Expense & Income Tracker built using vanilla JavaScript.

Live Demo
https://diptendudas-expense-tracker.netlify.app

Features

- Add, edit, delete expenses
- Add, edit, delete income
- Category management
- Filter by category and month
- Sort by date and amount
- Monthly budget tracking
- Expense visualization using Chart.js
- Dark mode with persistent theme
- LocalStorage-based state management
- Fully deployed on Netlify

Tech Stack

- HTML5
- CSS3
- JavaScript (ES Modules)
- Chart.js
- Netlify (Deployment)

What I Learned

- Modular JavaScript architecture
- DOM manipulation & state handling
- Data persistence using localStorage
- Theme toggling using class-based CSS
- Static site deployment
